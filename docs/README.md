# 文档导航

这份文档现在只做总入口，不再把所有链接都铺在根目录。

如果不知道该从哪开始，按下面顺序走：

1. 先看 [归档/README.md](归档/README.md) 衔接最近会话和推进记录
2. 再看 [长期说明/README.md](长期说明/README.md) 找架构、配置、差异和工作流
3. 需要按项执行时看 [清单/README.md](清单/README.md)
4. 需要看模块专项时看 [modules/README.md](modules/README.md)
