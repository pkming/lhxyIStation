# 图片对照整理说明

这批图片不是废图，绝大多数都有保留价值。

现在统一按下面这条规则使用：

1. 主图源以 `docs/设备手册/M90/README.md` 和同目录 `副本AL-M90车载智能终端使用说明书20250510_V1.1_CN.files` 为准
2. `docs/图片对照/手册原图归档` 保存按用途整理过的完整原图归档
3. `docs/图片对照` 其余目录只保留精选后的快捷入口，方便快速查配置页和联调页面
4. 后面零散补拍的照片不再作为主依据，除非手册原图里没有对应页面

原因很直接：`副本AL-M90车载智能终端使用说明书20250510_V1.1_CN.files` 里的图更全，和当前图片对照里的核心页面是同一套内容，只是覆盖范围更大、准确性更高。

它们现在主要服务 3 件事：

1. 给新版 M90 页面和旧逻辑做界面对照
2. 给报站、GPS、网络、外设、DVR 联调提供配置基线
3. 在没有反编译的前提下，先确认终端真实可见的功能面板和关键参数

3. 做页面对照时先定位，再回手册原图看整组内容
4. 需要按用途批量查图时，优先打开 `手册原图归档`

### 01-主界面与报站

用途：

1. 核首页按钮布局和状态区域
2. 对照 DVR 画面、GPS 状态、网络状态在主界面的落点
3. 作为后续首页直达报站行为对齐的视觉依据

### 02-线路与调度

用途：

1. 核线路选择页面结构
2. 核调度中心预置短语入口
3. 判断线路属性展示方式是不是要单独做模块页

### 03-网络与版本

用途：

1. 核平台服务器参数
2. 核心跳、信息上报、超速上报间隔
3. 核系统版本、数据库版本
4. 核 GPS/LAN/4G/WIFI/服务器连接状态展示

### 04-外设与串口

用途：

1. 核 RS232 / RS485 / GPS / DVR 的真实配置项
2. 核旧屏显协议可选项列表
3. 核 GPS 波特率和 DVR 通道模式

### 05-报站设置与音量语言

用途：

1. 核报站设置页真实开关项
2. 核 TTS 车内车外音量项
3. 核语言设置项

### 06-站点学习

用途：

1. 核线路列表和站点学习页面结构
2. 核站点 GPS、实时 GPS、角度、速度字段
3. 核 OTHER LEARN 和 remind 属性入口

### 07-DVR原生菜单

用途：

1. 保留原生 DVR 菜单能力边界
2. 区分哪些功能属于业务壳，哪些属于 DVR 自身菜单

## 直接可用的结论

下面这些信息已经可以直接拿来指导新壳迁移和联调：

### 外设与串口

从 [04-外设与串口/外设设置-RS485-2-JHY-DVR通道-VIN4-AUTO-GPS115200.jpg](04-%E5%A4%96%E8%AE%BE%E4%B8%8E%E4%B8%B2%E5%8F%A3/%E5%A4%96%E8%AE%BE%E8%AE%BE%E7%BD%AE-RS485-2-JHY-DVR%E9%80%9A%E9%81%93-VIN4-AUTO-GPS115200.jpg) 和同组图片可确认：

1. `RS232-1` 走 `DVR`，波特率 `9600`
2. `RS232-2` 默认可为 `无`，波特率 `9600`
3. `RS485-1` 波特率 `38400`
4. `RS485-2` 可见 `JHY` 协议，波特率 `9600`
5. 旧屏显协议项至少出现：`通达`、`恒舞`、`武汉乐的`、`海梁`、`LED导程牌`、`LHXY`、`HW3`、`JHY`
6. `GPS` 波特率可见 `115200`
7. `DVR CHANNEL` 至少出现 `VIN4-AUTO`，补拍图里还有 `VIN1-4-AUTO`

### 网络与平台

从 [03-网络与版本/网络设置-服务器地址-端口-心跳信息超速间隔.jpg](03-%E7%BD%91%E7%BB%9C%E4%B8%8E%E7%89%88%E6%9C%AC/%E7%BD%91%E7%BB%9C%E8%AE%BE%E7%BD%AE-%E6%9C%8D%E5%8A%A1%E5%99%A8%E5%9C%B0%E5%9D%80-%E7%AB%AF%E5%8F%A3-%E5%BF%83%E8%B7%B3%E4%BF%A1%E6%81%AF%E8%B6%85%E9%80%9F%E9%97%B4%E9%9A%94.jpg) 和 [03-网络与版本/网络设置-协议下拉-ALINK-CC808.jpg](03-%E7%BD%91%E7%BB%9C%E4%B8%8E%E7%89%88%E6%9C%AC/%E7%BD%91%E7%BB%9C%E8%AE%BE%E7%BD%AE-%E5%8D%8F%E8%AE%AE%E4%B8%8B%E6%8B%89-ALINK-CC808.jpg) 可确认：

1. 调度平台服务器示例 IP 为 `123.60.61.0`
2. 调度平台端口示例为 `50804`
3. 心跳间隔 `10秒`
4. 信息间隔 `15秒`
5. 超速间隔 `5秒`
6. 文件更新服务器示例 IP 为 `39.104.66.194`
7. 文件更新服务器端口示例为 `8081`
8. 网络协议下拉至少出现：`无`、`ALINK`、`CC808`

### 报站设置

从 [05-报站设置与音量语言/报站设置-线路属性-对开环线防反-方言外音整点报时.jpg](05-%E6%8A%A5%E7%AB%99%E8%AE%BE%E7%BD%AE%E4%B8%8E%E9%9F%B3%E9%87%8F%E8%AF%AD%E8%A8%80/%E6%8A%A5%E7%AB%99%E8%AE%BE%E7%BD%AE-%E7%BA%BF%E8%B7%AF%E5%B1%9E%E6%80%A7-%E5%AF%B9%E5%BC%80%E7%8E%AF%E7%BA%BF%E9%98%B2%E5%8F%8D-%E6%96%B9%E8%A8%80%E5%A4%96%E9%9F%B3%E6%95%B4%E7%82%B9%20%E6%8A%A5%E6%97%B6.jpg) 可确认：

1. 线路属性有 `对开`、`环线`、`防反`

3. 有 `是否播报方言`
4. 有 `是否播报英文`
5. 有 `是否开外音`
6. 有 `是否整点报时`

### 站点学习

从 [06-站点学习/站点学习-线路列表-BTC-SGHB-SGHO.jpg](06-%E7%AB%99%E7%82%B9%E5%AD%A6%E4%B9%A0/%E7%AB%99%E7%82%B9%E5%AD%A6%E4%B9%A0-%E7%BA%BF%E8%B7%AF%E5%88%97%E8%A1%A8-BTC-SGHB-SGHO.jpg)、[06-站点学习/站点学习-站名-GPS-角度-速度.jpg](06-%E7%AB%99%E7%82%B9%E5%AD%A6%E4%B9%A0/%E7%AB%99%E7%82%B9%E5%AD%A6%E4%B9%A0-%E7%AB%99%E5%90%8D-GPS-%E8%A7%92%E5%BA%A6-%E9%80%9F%E5%BA%A6.jpg) 和 [06-站点学习/站点学习-OTHER-LEARN-remind属性-GPS坐标.jpg](06-%E7%AB%99%E7%82%B9%E5%AD%A6%E4%B9%A0/%E7%AB%99%E7%82%B9%E5%AD%A6%E4%B9%A0-OTHER-LEARN-remind%E5%B1%9E%E6%80%A7-GPS%E5%9D%90%E6%A0%87.jpg) 可确认：
