# M90 手册原图归档

这份归档把 M90 新手册原图从编号文件整理成可直接查阅的分类目录。

主图源来自：

1. [../../设备手册/M90/README.md](../../设备手册/M90/README.md)
2. `../../设备手册/M90/副本AL-M90车载智能终端使用说明书20250510_V1.1_CN.files`

整理原则：

1. 按页面用途分组，不按导出编号堆放
2. 文件名改成可读中文名，便于联调和引用
3. 手册 README 未引用但实际存在的图片单独放到 `99-未在README引用`
