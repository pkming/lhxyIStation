# 模块文档导航

这份目录只收模块专项文档。

如果现在的问题不是“先看哪份模块文档”，而是“看完文档后代码该进哪里”，先回：

1. [../长期说明/文档链路与代码链路总览.md](../长期说明/文档链路与代码链路总览.md)

根目录 `docs/` 继续放：

- 总体推进说明
- 架构说明
- 配置与工作流
- 联调清单
- 历史差异记录

模块专属文档统一收在 `docs/modules/`，避免 GPS、报站、DVR 说明继续散在根目录。

## 当前已整理模块

### GPS

先看：

1. [gps/README.md](gps/README.md)
2. [gps/GPS页面图册.md](gps/GPS页面图册.md)

适用场景：

1. 要核 GPS 基础链路是否稳定
2. 要看 L1 基线和完整迁移顺序
3. 要确认当前 GPS 模块整理到了哪一步

### 报站

先看：

1. [station/README.md](station/README.md)
2. [station/报站页面图册.md](station/报站页面图册.md)

适用场景：

1. 要核旧版报站链路和新壳迁移步骤
2. 要继续推进报站与 GPS 自动报站联动
3. 要查旧版方法名和新壳落点
