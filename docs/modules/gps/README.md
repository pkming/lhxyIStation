# GPS 模块文档

这组文档只服务 GPS 模块整理和完整迁移。

## 阅读顺序

1. [GPS校验文档.md](GPS校验文档.md)
2. [GPS迁移路线图.md](GPS迁移路线图.md)
3. [GPS页面图册.md](GPS页面图册.md)

## 各文档职责

### GPS校验文档

适合：

1. 真机前核 GPS 串口、定位状态、DVR GPS 上报
2. 选一条基线线路做完整复刻
3. 判断当前版本距离旧项目完整 GPS 功能还差什么

### GPS迁移路线图

适合：

1. 开始拆完整 GPS 功能迁移任务
2. 看旧项目 GPS 功能在新壳里的目标落点
3. 按阶段推进自动报站、友情提醒、校时、学习落库

### GPS页面图册

适合：

1. 核 GPS 状态位、站点学习、网络配置、天线接线
2. 给 GPS 联调和页面核对提供直接图源
3. 把 GPS 行为和真实终端页面一一对应起来
