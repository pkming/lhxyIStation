### 旧版报站链路说明

1. 查旧项目 MainActivity / Presenter 的链路入口
2. 查旧版方法名、文件名和依赖字段
3. 做“旧逻辑到底在哪里”的代码考古

### 新版报站迁移步骤

适合：

1. 看新壳当前已经迁到哪里
2. 看报站中心当前执行步骤
3. 继续补语音、TTS、GPIO 和首页直达节奏

### 报站页面图册

适合：

1. 按页面核首页报站、重复、切换、停止的真实入口
2. 核报站设置、TTS、语言、音量页面
3. 查报站资源目录、CommonSounds、外音序列的原图依据
